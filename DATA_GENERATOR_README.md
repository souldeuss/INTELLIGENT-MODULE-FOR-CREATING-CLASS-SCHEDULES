# 🎲 Генератор тестових даних

## Опис

Функціонал автоматичної генерації рандомних даних для тестування системи розкладу. Включає генерацію:

- ⏰ Таймслотів (5 днів × 6 періодів)
- 🏫 Аудиторій (лекційні, комп'ютерні класи, лабораторії)
- 👨‍🏫 Викладачів (з українськими іменами)
- 👥 Студентських груп
- 📚 Курсів

### Backend API

#### Генерація даних

```bash
POST /api/seed/generate

# З параметрами
POST /api/seed/generate?num_teachers=10&num_groups=8&num_courses=20&num_classrooms=15
```

**Параметри:**

- `num_teachers` - кількість викладачів (за замовчуванням: 8)
- `num_groups` - кількість груп (за замовчуванням: 6)
- `num_courses` - кількість курсів (за замовчуванням: 14)
- `num_classrooms` - кількість аудиторій (за замовчуванням: 12)

**Відповідь:**

```json
{
  "timeslots": 30,
  "classrooms": 12,
  "teachers": 8,
  "groups": 6,
  "courses": 14,
  "message": "Рандомні дані успішно згенеровані! ✅"
}
```

#### Очищення всіх даних

```bash
DELETE /api/seed/clear
```

**Відповідь:**

```json
{
  "message": "Всі дані успішно видалені"
}
```

## Особливості

1. **Унікальність імен**: Система генерує унікальні імена викладачів
2. **Транслітерація**: Email-адреси автоматично створюються з транслітерацією українських імен
3. **Реалістичні дані**: Використовуються реальні українські імена, прізвища та по батькові
4. **Очищення перед генерацією**: Якщо дані вже існують, вони автоматично видаляються
5. **Правильні зв'язки**: Курси автоматично прив'язуються до відповідних типів аудиторій
